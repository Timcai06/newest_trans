# 谷歌翻译插件视觉升级样式文档 (Style Guide)

本文档详细说明了本次视觉升级的设计理念、修改细节及技术实现方案。

## 1. 设计理念 (Design Philosophy)

本次升级旨在打造**现代、流畅且富有深度**的用户界面，核心关键词为：
- **Glassmorphism (毛玻璃拟态)**: 利用背景模糊 (`backdrop-filter`) 创造层次感。
- **Fluidity (流动性)**: 通过平滑的过渡动画和流体渐变增强交互体验。
- **Consistency (一致性)**: 统一全站的色彩、圆角、阴影和交互反馈。

## 2. 核心组件升级 (Core Component Upgrades)

我们对以下 5 类核心 UI 元素进行了深度视觉升级：

### 2.1 翻译记录卡片 (`.word-item`)
- **流体边框动画**: 
  - 实现方案: 使用 `.word-border` 容器配合 `conic-gradient` 和 `mask-composite`。
  - 视觉效果: 鼠标悬停时，七彩渐变光流沿着卡片边缘流动，形成空心发光边框效果。
  - **自适应对比度**: 浅色卡片使用深色光谱（Deep Spectrum）增强可见度，深色卡片使用明亮光谱（Bright Neon）产生霓虹质感。
  - 技术细节: `mask-composite: exclude` 确保中间镂空，只显示边框。
- **卡片质感**: 
  - 默认状态: 全透明背景，依靠阴影和细微边框勾勒轮廓。
  - 悬停状态: 背景变为高斯模糊白 (`rgba(255,255,255,0.85)`)，提升内容可读性。

### 2.2 首页统计卡片 (`.stat-card`)
- **交互反馈**: 
  - 悬停时整体上浮 (`translateY(-5px)`) 并增强阴影。
  - 图标产生微缩放和旋转动画 (`scale(1.2) rotate(5deg)`).
- **视觉层次**: 
  - 采用渐变文字显示数字，增强视觉冲击力。
  - 添加微弱的背景光效 (`linear-gradient`) 增强立体感。

### 2.3 学习面板按钮 (`.learning-btn`)
- **统一风格**: 
  - 统一使用圆角矩形 (`border-radius: 16px`)。
  - 默认半透明玻璃质感，悬停时填充主色调 (`--accent-primary`)。
- **状态反馈**: 
  - 点击时有缩放反馈 (`active` 状态)。
  - 文字颜色在悬停时自动反转为白色，确保对比度。

### 2.4 控制栏与输入框 (`.controls`, `.search-input`)
- **现代感增强**: 
  - 移除默认的厚重边框，改为细边框 + 内部光晕。
  - 聚焦时产生柔和的主色光环 (`box-shadow`).
  - 统一所有功能按钮（导入/导出/设置）的高度和圆角。

### 2.5 滚动条 (`::-webkit-scrollbar`)
- **细节优化**: 
  - 宽度缩减为 8px，视觉更轻量。
  - 滑块支持深色模式适配，悬停时加深颜色。
  - 轨道背景半透明，不遮挡底层背景纹理。

## 3. 双重主题系统 (Dual Theme System)

我们简化了主题系统，只保留两种风格迥异的对立设计：

### 3.1 浅色模式 (Light Mode)
- **风格定位**: 清新、专业、通透。
- **视觉特征**: 
  - 背景: 浅色渐变或纯白。
  - 卡片: 乳白磨砂 (`rgba(255,255,255,0.9)`), 极致纯净。
  - 光效: **深色光谱 (Deep Spectrum)** - 使用更深邃的靛蓝、深红、墨绿，在白色卡片外围形成强烈反差。
  - 边框: 宽度加倍至 **4px**，视觉冲击力大幅提升。

### 3.2 深色模式 (Dark Mode)
- **风格定位**: 沉浸、霓虹、赛博朋克。
- **视觉特征**: 
  - 背景: 深邃的 Slate 灰蓝 (`#0f172a`)。
  - 卡片: 极夜黑 (`rgba(2,6,23,0.9)`), 接近不透明，为光效提供最黑的画布。
  - 光效: **明亮光谱 (Bright Neon)** - 使用高亮紫、荧光粉、青色，如同黑夜中燃烧的霓虹灯管。
  - 边框: 宽度加倍至 **4px**，光晕感更强。

通过 CSS 变量 (`:root` vs `body.theme-dark`) 实现无缝切换，确保每一处细节（文字、边框、阴影、光晕）都完美适配当前氛围。

## 4. 动画实现细节 (Animation Details)

### 边框旋转动画
```css
@keyframes borderRotate {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}
```
应用在 `.word-border::before` 上，配合 200% 尺寸的圆锥渐变背景，实现无缝循环流动。

### 性能优化
- 所有动画均使用 `transform` 和 `opacity` 属性，确保触发 GPU 加速。
- 避免触发布局重排 (Layout Thrashing)，保证 60fps 流畅度。

## 5. 维护说明
- 所有新样式均在 `popup.css` 中定义，并带有清晰的注释区块。
- 修改主题颜色只需调整 `:root` 下的 CSS 变量即可全局生效。
